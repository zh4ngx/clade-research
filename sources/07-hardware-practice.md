## Platform Calibration Guide for Genmitsu Cubiko

**YouTube ID:** TiOBManW8CY
**Topic:** Machine Calibration / Z-Probe

### Summary
Step-by-step guide to calibrating the Genmitsu Cubiko CNC platform using the height map alignment system. This process ensures accurate Z-height detection across the work surface, which is essential for precise engraving and cutting operations.

### Key Steps/Points
1. Take out the height map alignment cable
2. Loosen the flat nose screw on the spindle
3. Insert the fork port into the screw and tighten
4. Connect the cable to the leveling port
5. Select "Calibration" in the app
6. Click "Z probe" and navigate through next prompts
7. Machine will home and move to the Z-probe module
8. Wait until stopped, then clamp the clip onto the bit
9. Click next to start detecting platform height
10. Wait for measurement to complete (shows successful)
11. Click confirm to complete the process
12. **Important:** Remove the height map alignment cable before beginning any project

### Tools/Parts Mentioned
- Height map alignment cable
- Fork port connector
- Z-probe module
- Genmitsu Cubiko mobile app

## Connecting Genmitsu APP with Cubiko CNC

**YouTube ID:** xZCwPs6HVC0
**Topic:** Wi-Fi Setup / App Connection

### Summary
How to connect the Cubiko CNC machine to the Genmitsu mobile app via Wi-Fi for wireless control and project management.

### Key Steps/Points
1. Connect to Cubiko Wi-Fi network
2. Open the Genmitsu app
3. Click "connecting Wi-Fi"
4. Go to network settings
5. Connect to the Wi-Fi network you want Cubiko to use
6. Switch your phone's Wi-Fi to the same network
7. Return to the app and click the Wi-Fi icon
8. Click the right arrow icon
9. Confirm settings
10. Move spindle to Z-probe module
11. Set probe settings (enter zero in G92Z parameters)
12. Enter material thickness in GZ parameters
13. Start probe and wait for stop, click Z then confirm
14. Raise Z-axis to prevent collisions
15. Move XY-axis to starting position
16. Click X0 and Y0 to set origin points
17. Open file, load your project, and run

### Tools/Parts Mentioned
- Genmitsu mobile app
- Wi-Fi network
- G-code parameters (G92Z, GZ)

## Installing 20K Spindle Motor on Genmitsu Cubiko

**YouTube ID:** 24gPqRLlwcs
**Topic:** Spindle Upgrade / Tool Installation

### Summary
Complete walkthrough for upgrading the standard spindle to a 20,000 RPM spindle motor for faster and more precise cutting operations.

### Key Steps/Points
1. Loosen four fixing screws
2. Remove the spindle cover
3. **Safety first:** Turn off the machine
4. Disconnect the spindle wire
5. Loosen the spindle screw
6. Remove the old spindle
7. Install the 20,000 RPM spindle
8. Tighten the spindle screw
9. Connect red cable to M+, black cable to M-
10. Install the cover back
11. Tighten four cover screws
12. Unscrew the collet nut
13. Push the collet into the collet nut toward the arrow until a click is heard
14. Install the bit
15. Tighten the collet

### Tools/Parts Mentioned
- 20,000 RPM spindle motor
- Collet and collet nut
- Fixing screws (4)
- Phillips screwdriver
- Bits appropriate for collet size

## Learning Path

Recommended order for learning CNC/hardware fabrication with the Genmitsu Cubiko:

### Phase 1: Getting Started
1. **Unboxing Cubiko** (mqfZuoovHqQ) - Understand what comes in the box
2. **Shop Smart, Create More** (OrBuLdgEJfM) - Get overview of capabilities
3. **Connect Genmitsu App** (xZCwPs6HVC0) - Set up wireless control

### Phase 2: Essential Calibration
4. **Platform Calibration Guide** (TiOBManW8CY) - Critical first step for accuracy

### Phase 3: First Projects
5. **Start Engraving** (mo19oI7Rx7U) - Your first CNC project

### Phase 4: Module Installation (as needed)
6. **Installing Dust Shoe** (6CMrvUWTttk) - For cleaner operation
7. **Installing 20K Spindle** (24gPqRLlwcs) - For faster/more precise cutting
8. **Installing Drawing Tool** (izoRDTeJgXQ) - For pen plotting
9. **Installing Laser Module** (7sfsdD-I3LQ) - For laser engraving
10. **Connect Laser Engraver App** - For laser-specific control

### Tips for Success
- Always turn off machine before installing/removing modules
- Remove calibration cable before running projects
- Raise Z-axis before moving XY to prevent collisions
- Always check origin points (X0, Y0, Z0) before running
- Use 3D preview to verify project placement
- Start with softer materials (wood, plastic) before metals
- Wear appropriate safety equipment (eye protection, dust mask)

### Common G-Code Commands Referenced
- `G92Z` - Set current Z position as zero
- `GZ` - Material thickness parameter
- `X0`, `Y0` - Set X and Y origin points
